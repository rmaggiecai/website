Brit + Co. Infinite Scroll Tab Redesign

Disclaimer: this design was not implemented by Brit + Co.

About the Project

Infinite scroll at the bottom of each article was introduced to create a continuous reading experience. Since there are limited number of related- reads, the infinite scroll also contains popular reads and latest posts so that users will never reach a dead end.

The tabs indicate what type of articles are being displayed. Users can also filter through the different types of articles by clicking on the tabs. However, some users get confused when using these tabs.

Using heuristic evaluation by Jakob Nielsen, I identified 3 problems with the original design:

#1 Tab menu design does not match real-world mental models

There is no UI element to visually connect the active tab with content displayed, unlike section dividers used in notebooks and binders.

#2 Active colors for inactive tabs

Tabs are coded in bright colors at all times. The lack of contrast make it difficult for users to differentiate active tab from inactive tabs.

#3 Inconsistent feedback

The hover state for these tabs do not have accompanied animation while other clickable features on the website do.

Proposed Design

After looking at different tab-style navigations, I came up with this final design. I introduced a horizontal line in the same color as the active tab. This line visually connects the content displayed with the active tab. I made the inactive tabs smaller to create depth. This illusion helps the active tab stand out from the inactive tabs. It also creates layers, similar to the way dividers stack inside a binder. To further differentiate active tab from inactive tabs, only the active tab is in color.

I wanted to keep the hover state animation consistent, thus it resembles those used for the main navigation bar and the social icons.
